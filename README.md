# checkBadWord
敏感词检测，使用tp5框架

# 运行环境
- PHP >= 5.6 (开发版本时的版本)
- thinkPHP 5.x.x
- Redis
- MySQL

# 安装
参考:[ ThinkPHP5.0完全开发手册](https://www.kancloud.cn/manual/thinkphp5/118003)

# 使用
很简单，没什么好讲的

# 注意事项
> 本案使用Redis缓存，若没有安装Redis请修改TP配置文件`\application\config.php`
